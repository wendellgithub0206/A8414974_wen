# 貪食蛇

## 規則
    蛇撞到牆壁以及蛇身及失敗
    分數計算:每吃一個食物獲得10分
    開始遊戲請按1
### 編碼
使用大量2D [`Vector`](https://docs.microsoft.com/zh-tw/cpp/standard-library/vector-class?view=vs-2019)<br />
使用讀取鍵盤[`conio`函式](https://www.runoob.com/w3cnote/c-get-keycode.html)<br />
簡單條件運算子<br />
拆成三個檔案  snake.h 、  snake.cpp  、 main.cpp<br />
